JoJo's Shave Ice - Task Setup System

Set up daily tasks for your locations

Welcome! This utility manages tasks for your business groups and locations.

Tasks are organized with this structure:

  • Business Groups (e.g., "Chicken in a Barrel", "JoJo's Shave Ice")
  • Locations within each group (e.g., "Waimea", "Coconut Marketplace")
  • Tasks for each location

Data Structure

Business Groups

Each business group (e.g., Chicken in a Barrel) can have:

  • groupName: Display name
  • groupCode: Short code
  • toastApiKey: For POS integration
  • active: Status flag
Locations

Each location (e.g., Waimea) within a group has:

  • locationName: Display name
  • locationCode: Short code
  • address: Physical address
  • locationEmail: For authentication
  • active: Status flag
Tasks

Tasks are linked to specific locations within groups:

  • groupId: References the business group
  • locationId: References the specific location
  • category, type, name, details: Task attributes
